# 🧠 Proyecto Clasificación

## Descripción General

Este proyecto tiene como objetivo principal desarrollar un sistema de **clasificación automática** que, a partir de ciertos parámetros de entrada, sea capaz de predecir una clase o categoría de salida. Está diseñado para resolver un problema de toma de decisiones en base a datos históricos.

## 🎯 Objetivo

- Implementar un algoritmo de clasificación (ej. Árboles de Decisión, SVM, o Redes Neuronales).
- Optimizar el modelo mediante técnicas de búsqueda de hiperparámetros (GridSearchCV).
- Generar predicciones con un alto grado de confianza y generalización.
